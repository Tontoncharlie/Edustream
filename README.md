# Student Management Dashboard UI

A modern, responsive student management dashboard built with Next.js and Tailwind CSS.

## Features
- Responsive layout for all screen sizes
- Dark mode support
- Clean and modern UI
- Dashboard navigation with icons
- Scalable frontend architecture

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- JavaScript
